gskat by kongo2002

A Gtk implementation of the popular german card game "Skat"
gskat allows you to play Skat against two computer controlled players. Since the game is currently in beta state you cannot expect too much AI from the computer opponents. Nevertheless they strictly follow the game rules, do not cheat and act on some major game playing principles.

Installation

The release packages are shipped with all needed resources like card images and icons whereas the development archives contain the version-controlled source code only.

Releases

Development

There are different source versions of gskat to download. Either choose one of the tagged versions or the latest development snapshot from github:
  • Version 0.0.2
  • Version 0.0.1
  • Latest development snapshot (CAUTION: might be unstable)

Dependencies

  • >= glib-2.16
  • >= gtk+-2.18

Contribute

You can contribute to gskat in several ways:
  • Request new features or report errors and bugs via the issue tracker
  • Translate gskat into your language

Moreover you can clone the project with git by running: 

$ git clone git://github.com/kongo2002/gskat
